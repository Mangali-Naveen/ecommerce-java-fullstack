# E-Commerce Java Full Stack Application

A modern full-stack E-Commerce application built with React and Spring Boot.

## Tech Stack

### Frontend

* React.js
* React Router
* Redux Toolkit
* Tailwind CSS
* Axios

### Backend

* Spring Boot
* Spring Security
* JWT Authentication
* Spring Data JPA (Hibernate)
* MySQL

## Features

* User Registration & Login
* JWT Authentication & Authorization
* Product Management (Admin)
* Product Listing & Search
* Shopping Cart
* Order Management
* Admin Dashboard
* Responsive UI

## Project Structure

ecommerce-java-fullstack/

* client/ (React Frontend)
* backend/ (Spring Boot Backend)

## Status

Frontend setup completed. Spring Boot backend development in progress.
